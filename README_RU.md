# Ветка с переводами



Как установить?<br />
Нажмите зелёную кнопку "Code", затем "Download Zip".<br />
Распакуйте загруженную папку "slave-of-gmod-modified-translations" в папку *addons* (steam\steamapps\common\GarrysMod\garrysmod\addons)

**Не стесняйтесь делать _pull request_ с новыми переводами.**
