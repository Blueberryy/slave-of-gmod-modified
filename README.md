# Translations branch



How to install?<br />
Click green "Code" button then "Download Zip".<br />
Extract downloaded "slave-of-gmod-modified-translations" folder to your *addons* folder (steam\steamapps\common\GarrysMod\garrysmod\addons)

**Feel free to _pull request_ new translations.**
